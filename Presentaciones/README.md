## Presentaciones de Ponencias

Este directorio contiene las presentaciones utilizadas en las ponencias del congreso. Cada archivo debe estar nombrado según el título de la ponencia y el nombre del ponente para facilitar su identificación.

Si tienes una presentación que deseas compartir, por favor súbela a este directorio siguiendo el formato establecido.

## Presentaciones on-line

Si tu presentación está on-line (por ejemplo, en GitHub pages, Google Slides, Prezi, etc.), puedes incluir un archivo de texto con el enlace a la presentación junto con una breve descripción. También puedes añadir el enlace a este README para que otros puedan acceder fácilmente.
